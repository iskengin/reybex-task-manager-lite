# PWA Icons

Bu klasöre PWA için gerekli icon dosyalarını ekleyin:

- `icon-192x192.png` - 192x192 piksel PNG icon
- `icon-512x512.png` - 512x512 piksel PNG icon  
- `apple-touch-icon.png` - 180x180 piksel Apple touch icon

## Icon Oluşturma

1. 512x512 piksel bir logo/icon hazırlayın
2. Online araçlar kullanarak (örn: https://realfavicongenerator.net/) farklı boyutlarda iconlar oluşturun
3. Oluşturulan iconları bu klasöre ekleyin

Not: Icon dosyaları olmadan da uygulama çalışır, ancak PWA kurulumunda icon görünmez.
